# Ololade Portfolio Site

A React + Vite + Tailwind portfolio site prepared for GitHub and Netlify deployment.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Deploy to Netlify

- Push this folder to GitHub
- In Netlify, import the GitHub repo
- Build command: `npm run build`
- Publish directory: `dist`

## Notes

- Navigation is hash-based, so it works cleanly on Netlify free hosting.
- Update the email address in `src/App.jsx` if needed.
- You can replace the accent color `#312e81` with another brand color later.
